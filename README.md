# FastAPI AI Chatbot

A secure AI chatbot backend built using FastAPI, SQLAlchemy, SQLite, JWT Authentication, User Management, Chat History, and Gemini AI Integration.

## Features

* User Registration
* User Login
* Password Hashing
* Email Validation
* Username Validation
* Secure Authentication
* JWT Token Authentication (In Progress)
* Protected Routes (In Progress)
* Chatbot Integration (In Progress)
* User Chat History (Planned)
* Google Authentication (Planned)

## Tech Stack

* FastAPI
* Python
* SQLAlchemy
* SQLite
* Pydantic
* JWT
* Gemini AI
* Git & GitHub
